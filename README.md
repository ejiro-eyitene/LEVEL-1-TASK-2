## Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a Stock Prices dataset.  

The EDA includes:

1. **Data Cleaning**
   - Checked for missing values and cleaned the dataset.
   - Dropped rows with missing values in critical columns (`Open`, `High`, `Low`) to ensure accurate analysis.
   - Cleaned column names to remove extra spaces or inconsistencies.

2. **Summary Statistics**
   - Mean, Median, Mode, Standard Deviation
   - Helps understand central tendency and variability of stock prices

3. **Data Visualization**
   - Histograms, Boxplots, Scatter plots
   - Understand distributions, outliers, and relationships between features

4. **Correlation Analysis**
   - Identify relationships between numerical features such as Open, Close, High, Low, and Volume

---

## Dataset
The dataset contains historical stock prices with features like:  

- `Open` – Opening price  
- `High` – Highest price of the day  
- `Low` – Lowest price of the day  
- `Close` – Closing price  
- `Volume` – Number of shares traded  
- `Date` – Trading date  

> Note: Rows with missing values in critical columns were removed to maintain accuracy.
